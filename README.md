### Go Fortnox client

Supported API:

#### 1. Invoices:

    - GET /invoices 
    - GET /invoices/:id
    - PUT /invpices/:id
    - POST /invoices

#### 2. Customers:

    - GET /customers
    - GET /customers/:id
    - PUT /customers/:id
    - POST /customers/:id
    - DELETE /customers/:id
